# 랜덤 자리 뽑기

로컬에서 실행하는 교실/스터디용 자리 배정 웹앱입니다.  
단순 랜덤 배정을 넘어서 `통로/비활성 좌석`, `고정석`, `명단 템플릿`, `이력 저장`, `선택 재추첨`, `직전 좌석 회피`, `앞/뒤·좌/우 균형 배정`, `복사/인쇄`, `이름 검색`까지 지원합니다.

## 기술 스택
- React
- TypeScript
- Vite
- Vitest
- localStorage

## 주요 기능
- `명단 입력`: 줄바꿈, 쉼표, 탭 입력을 모두 파싱
- `좌석판 편집`: 좌석 / 통로 / 비활성 셀 토글
- `학생 수 자동 맞춤`: 학생 수 기준 추천 행/열 조합 제안
- `고정석`: 특정 학생을 특정 좌석에 고정
- `랜덤 배정`: 사용 가능한 좌석만 대상으로 배정
- `선택 재추첨`: 일부 학생만 다시 뽑기
- `직전 좌석 회피`: 바로 이전 회차의 같은 좌석 우선 회피
- `균형 배정`: 앞/뒤, 좌/우 편중 완화
- `템플릿 저장`: 명단 + 좌석배치 + 고정석 + 통로 설정 저장
- `이력 저장`: 최근 20회 결과 보관
- `이름 검색`: 현재 자리표에서 학생 위치 하이라이트
- `복사/인쇄`: 텍스트 복사 및 브라우저 인쇄 지원

## 시작하기
### 요구 사항

- npm 10 이상 권장


```bash

```

## 사용 방법
1. 왼쪽 상단 입력 영역에 학생 이름을 입력하거나 CSV/엑셀 데이터를 붙여넣습니다.
2. 행/열을 조정하거나 추천 좌석판을 적용합니다.
3. 좌석판에서 각 칸을 클릭해 `좌석 -> 통로 -> 비활성` 순으로 바꿉니다.
4. 필요한 경우 학생과 좌석을 선택해 고정석을 지정합니다.
5. 전체 배정 또는 선택 재추첨 모드를 선택합니다.
6. 필요하면 `직전 좌석 회피`, `균형 배정` 옵션을 켭니다.
7. `랜덤 배정` 또는 `선택 재추첨`을 실행합니다.
8. 결과판에서 이름 검색, 텍스트 복사, 인쇄를 사용할 수 있습니다.

## 저장 방식
이 앱은 서버를 사용하지 않습니다. 모든 데이터는 브라우저 `localStorage`에 저장됩니다.

저장되는 데이터:
- 현재 작업 중인 초안
- 좌석 배치 정보
- 고정석 정보
- 저장된 템플릿 목록
- 최근 20회 배정 이력
- 검색어
- 선택 재추첨 대상

브라우저 데이터를 지우거나 다른 브라우저로 이동하면 저장 내용은 공유되지 않습니다.

```text
src/

  utils/

    templates.ts          템플릿 유틸
    history.ts            이력 관리
    export.ts             복사/출력용 텍스트 생성
```

## 배정 규칙
- `seat` 타입 셀만 실제 배정 대상입니다.
- `aisle`, `blocked` 셀은 배정 대상에서 제외됩니다.
- 고정석 학생은 먼저 배치되고, 나머지 학생만 랜덤 배정됩니다.
- `선택 재추첨`은 선택되지 않은 학생과 고정석을 유지합니다.
- `직전 좌석 회피`는 가능한 경우에만 적용되고, 불가능하면 예외적으로 같은 좌석이 허용됩니다.
- `균형 배정`은 최근 이력을 기준으로 앞/뒤와 좌/우 편중을 줄이는 휴리스틱입니다.
